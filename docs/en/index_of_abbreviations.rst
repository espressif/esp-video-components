Index of Abbreviations
=======================

.. glossary::

    ISP
      Image Signal Processor
    
      ISP（图像信号处理器）是成像系统中负责处理图像的专用处理器。除了硬件处理之外，还会进行不同类型的软件增强，以确保所有成像参数（例如对比度、增益、锐度、伽玛等）均正确无误。
    
    BLC
      Black Level Correction
    
      黑电平校正
    
    DPC
      Static and Dynamic Defective Pixel Correction
    
      静态与动态坏点校正

    BF
      Bayer Noise Reduction Filter
    
      拜耳域降噪

    LSC
      Lens Shading Correction
    
      镜头阴影校正
    
    Demosaic
      Demosaic processing
    
      插值处理，将 RAW 数据转换为 RGB 数据

    CCM
      Color Correction Matrix
    
      颜色校正矩阵

    Gamma RGB
      Gamma correction in RGB domain
    
      RGB 域的伽马校正

    CSC
      Color Space Conversion
    
      颜色空间空间转换，包括 RGB2YUV、YUV2RGB。

    CA
      Color Adjustment Module

      颜色调整模块，包括亮度 (Brightness)、对比度 (Contrast)、饱和度 (Saturation) 和色度 (Hue) 四个方面的颜色处理。
      
    sharpen
      image sharpening
    
      图像锐化增强，也称为边缘增强 (Edge Enhancement, EE)。

    CLUT
      Color Look-Up Table enhancement (CLUT)
    
      颜色查表增强，包括对比度 (Contrast)、色度 (Hue)、饱和度 (Saturation)

    AE
      Auto Exposure

      自动曝光
    
    AEC
      Auto Exposure Control

      自动曝光控制

    AF
      Auto-Focus

      自动对焦
    
    AWB
      Auto White Balance

      自动白平衡  
    
    3A
      Auto Exposure, Auto-Focus, Auto White Balance

      自动曝光、自动对焦、自动白平衡
    
    fps
      Frames Per Second
      
      帧率

    IOCTL
      Input Output Control

      输入输出控制
    
    V4L2
      Video for Linux-2
    
      Linux 系统的标准视频设备的驱动框架

    DVP
      Digital Video Port

      相机并口传输协议，也称为 Parallel 口
    
    UVC
      USB Video Class

      USB 视频设备类

    MIPI
      Mobile Industry Processor Interface

      移动处理器接口

    CSI
      CMOS Sensor Interface

      相机串行传输接口

    SCCB
      Serial Camera Control Bus

      串行相机控制总线，兼容 I2C 总线

    CCI
      Camera Control Interface

      相机控制接口，与 SCCB 类似，可以通过 I2C 协议实现

    IQ
      Image Quality

      图像质量

    PCLK
      Pixel Clock

      传感器输出的像素时钟

    M2M
      Memory-to-Memory

      内存到内存的视频编码器接口
    
    IPA
      Image Processing Algorithms

      图像处理算法
    
    ROI
      Region Of Interest

      感兴趣的区域
    
    VCM
      Voice Coil Motor

      音圈马达
